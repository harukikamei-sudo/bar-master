# BAR MASTER - タスク進捗管理

## プロジェクト概要
3Dバーカウンター AIマスターお酒レコメンドWebアプリ
- 技術: React + Three.js (R3F)
- データ: JSON (フロントエンド完結、API不使用)
- ホスティング: 静的サイト (Vercel/Netlify)
